class と id とスタイルシート

HTML の中に物理要素（色とか、サイズとか）を書き込むのは、ルール違反です。 物理要素は、カスケーディングスタイルシート（スタイルシート）を使って記述します。

スタイルシートは、head の中に直接書くこともありますが、 普通は別ファイルで用意しておいて、head で読み込みます。

スタイルシートの中では、タグを指定する他、 ピリオド＋クラス名、ナンバー＋ID名で、クラスや ID を指定できます。
